Инструкция по использованию админ-панели MovieLunch.ru

Введение

Данная инструкция описывает процесс использования админ-панели для управления содержимым сайта MovieLunch.ru и редактирования раздела "Ход работы".

Важно: Админ-панель доступна только авторизованным пользователям. По умолчанию создан пользователь с логином admin и паролем admin123. Рекомендуется сменить пароль после первого входа.

Доступ к админ-панели

Для доступа к админ-панели выполните следующие шаги:

  1. Перейдите по адресу http://localhost:5000 (при локальном запуске) или по адресу, предоставленному администратором.
  2. Введите имя пользователя и пароль в форму авторизации.
  3. Нажмите кнопку "Войти".

После успешной авторизации вы получите доступ к функциям управления сайтом.

Управление разделом "Ход работы"

Раздел "Ход работы" представляет собой интерактивный to-do список с задачами, сгруппированными по категориям.

Добавление новой задачи

  1. Перейдите на вкладку "Ход работы" в админ-панели.
  2. Заполните форму "Добавить новую задачу":
    • Название задачи - краткое описание задачи
    • Подробное описание - детальное описание задачи, которое будет отображаться при раскрытии карточки
    • Категория - выберите одну из категорий: Дизайн, Сайт, Яндекс.Бизнес или Стратегия
    • Срочность - выберите уровень срочности: Высокая, Средняя или Низкая
    • Срок выполнения - укажите дату, к которой задача должна быть выполнена
    • Статус - выберите текущий статус задачи: Ожидание, В процессе или Завершено
  3. Нажмите кнопку "Добавить задачу".

Редактирование задачи

  1. Найдите нужную задачу в списке соответствующей категории.
  2. Нажмите кнопку "Редактировать" рядом с задачей.
  3. В открывшемся модальном окне внесите необходимые изменения.
  4. Нажмите кнопку "Сохранить изменения".

Удаление задачи

  1. Найдите нужную задачу в списке соответствующей категории.
  2. Нажмите кнопку "Удалить" рядом с задачей.
  3. Подтвердите удаление в диалоговом окне.

Примечание: Все изменения в разделе "Ход работы" автоматически отображаются на основном сайте без необходимости перезагрузки страницы.

Управление контентом сайта

Админ-панель позволяет редактировать основные элементы контента на сайте.

Редактирование заголовков и описаний

  1. Перейдите на вкладку "Контент сайта" в админ-панели.
  2. Найдите нужный раздел (Главная страница, Анализ, Рекомендации и т.д.).
  3. Внесите необходимые изменения в соответствующие поля.
  4. Нажмите кнопку "Сохранить изменения" для каждого раздела.

Настройки безопасности

В целях безопасности рекомендуется регулярно менять пароль доступа к админ-панели.

Изменение пароля

  1. Перейдите на вкладку "Настройки" в админ-панели.
  2. В разделе "Изменить пароль" введите текущий пароль.
  3. Введите новый пароль и подтвердите его.
  4. Нажмите кнопку "Изменить пароль".

Важно: Используйте надежные пароли, содержащие буквы разного регистра, цифры и специальные символы. Минимальная длина пароля - 8 символов.

Техническая поддержка
